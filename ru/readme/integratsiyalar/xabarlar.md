# Сообщения



### Создание шаблона сообщений

Раздел сообщений позволяет создавать шаблоны для сообщений, отправляемых клиентам и пользователям, и управлять ими. Через этот раздел сохраняются заранее подготовленные тексты и используются автоматически или вручную в необходимых процессах.\
Этот раздел используется для ускорения процесса отправки сообщений, сохранения единого формата и стандартизации коммуникаций.

<figure><img src="../../.gitbook/assets/image (1).png" alt=""><figcaption></figcaption></figure>

Осуществляется вход в раздел Интеграции → Сообщения, и через кнопку **"Создать"** создается новый шаблон сообщения. В каждом шаблоне вводится текст сообщения и сохраняется для дальнейшего использования в различных процессах.

Этот раздел состоит из двух основных частей:

* **Шаблон сообщения** – список созданных сообщений
* **История сообщений** – история отправленных сообщений

Шаблоны сообщений предоставляют следующие возможности:

* Использование готовых текстов для быстрой отправки сообщений
* Автоматизация без повторного написания одинаковых сообщений
* Создание отдельных шаблонов для различных ситуаций (задолженность, напоминания и другие)

Через этот раздел можно оптимизировать процесс связи с клиентами и системно управлять отправкой сообщений.
