# Etiketka

<details open>

<summary>Печать этикеток</summary>

Функция Etiketka используется для подготовки и вывода печатных **ярлыков (этикеток)** для товаров. Внешний вид этикетки и отображаемые на ней данные настраиваются в **настройках**.

<figure><img src="../../.gitbook/assets/image (297).png" alt=""><figcaption></figcaption></figure>

Пользователь может зайти в раздел **Савdo → Etiketka** для подготовки и печати этикеток для товаров.

При создании этикетки сначала выбирается **тип этикетки (шаблон)**. Каждый шаблон определяет размер и внешний вид этикетки.

<figure><img src="../../.gitbook/assets/image (296).png" alt=""><figcaption></figcaption></figure>

Затем из списка товаров выбирается нужный товар.

Также можно настроить, будет ли отображаться цена или нет.

После добавления товаров система подготавливает **количество этикеток в соответствии с введенным количеством**. Пользователь может распечатать этикетки с помощью кнопки **Print**.

Таким образом, можно быстро и удобно подготовить и распечатать необходимое количество этикеток для товаров.

</details>
