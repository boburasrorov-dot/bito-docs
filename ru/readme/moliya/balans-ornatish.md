# Установка баланса

<details>

<summary>Установка баланса клиента</summary>

Через данный раздел можно вручную вводить балансы в разрезе клиентов и сравнивать их с существующими балансами. Этот процесс используется для корректировки расчетов и приведения их в соответствие с реальным состоянием.

В процессе установки баланса пользователь сначала выбирает нужного клиента, и система автоматически отображает его существующий (плановый) баланс. Затем пользователь вводит значение "факт" в соответствии с реальным состоянием. Система сравнивает это введенное значение с существующим балансом и автоматически вычисляет разницу между ними. Если разница существует, это указывает на необходимость корректировки баланса. После ввода всех данных через сохранение баланс обновляется, и расчеты в системе приводятся в соответствие с реальным состоянием.

<figure><img src="../../.gitbook/assets/image (414).png" alt=""><figcaption></figcaption></figure>

<figure><img src="../../.gitbook/assets/image (34).png" alt=""><figcaption></figcaption></figure>

Для установки баланса нажимается кнопка **Создать**, и в специальном окне появляется список клиентов.

По каждому клиенту доступны следующие столбцы:

* **Имя** – название клиента
* **План** – баланс в системе (расчетный)
* **Факт** – реальный (вводимый вручную) баланс
* **Разница** – разница между планом и фактом

Балансы вводятся в разрезе следующих валют:

* **Сум**
* **USD (и могут быть другие валюты)**

#### Ввод данных

* В столбец **Факт** вводится реальный баланс
* Система автоматически рассчитывает **Разницу**
* В нижней части отображаются общие показатели **Итого**

#### Сохранение

* **Сохранить** – сохраняет введенные балансы
* **Отменить** – отменяет операцию
