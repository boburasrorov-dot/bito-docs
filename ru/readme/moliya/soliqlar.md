# Налоги



### Налоги

Раздел «Налоги» позволяет создавать и управлять налоговыми ставками, применяемыми к товарам и услугам в системе. Данный раздел служит для автоматизации ценообразования, правильного расчета платежей и обеспечения точности финансовой отчетности.

Для добавления налога пользователь нажимает кнопку **"Создать"** и вводит необходимые данные.

#### Основные данные

На этом этапе задаются основные параметры налога:

* **Название** — наименование налога (например: НДС, Акциз и другие)
* **Ставка (%)** — процентная ставка налога, это значение определяет, какой процент добавляется к цене товара
* **Тип** — тип налога (например: включенный или другой тип), определяет способ применения налога

<figure><img src="../../.gitbook/assets/image (461).png" alt=""><figcaption></figcaption></figure>

#### Настройки применения

Здесь определяется, как будет работать налог:

* **Ручное назначение** — если включено, налог выбирается вручную для каждой операции, не применяется автоматически
* **Включать в цену** — определяет автоматическое включение налога в цену товара

<figure><img src="../../.gitbook/assets/image (462).png" alt=""><figcaption></figcaption></figure>

#### Область применения

Выбирается, к каким товарам относится налог:

* **Все товары** — налог применяется ко всем товарам
* **Некоторые товары** — применяется только к выбранным товарам (выбираются через список ниже)

#### Привязка к товарам

Если налог применяется только к определенным товарам, доступны следующие возможности:

* поиск и добавление товаров
* выбор по категориям
* фильтрация по поставщику
