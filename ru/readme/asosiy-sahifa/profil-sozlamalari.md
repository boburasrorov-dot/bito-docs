# Настройки профиля

<details>

<summary>Редактирование личных данных</summary>

Для входа в окно профиля нажмите на имя пользователя в правом верхнем углу системы. В открывшемся меню выберите раздел **Профиль**.

В разделе профиля

* Загрузить свою фотографию
* Редактировать имя и фамилию
* Изменить номер телефона
* Дата рождения
* ИНН
* Telegram
* Email

Можно ввести данные и сохранить

<summary>Изменение пароля (в авторизованном состоянии)</summary>

Пользователь может обновить свой пароль после входа в систему.

**Порядок изменения пароля:**

1. В окне профиля выберите раздел **Изменение пароля**.
2. Введите действующий пароль.
3. Введите новый пароль.
4. Подтвердите новый пароль.
5. Нажмите кнопку **Сохранить**.

Для безопасности рекомендуется использовать надежный пароль.

<summary>Изменение PIN-кода (для мобильного приложения)</summary>

Для быстрого входа через мобильное приложение используется PIN-код.

#### Порядок изменения PIN-кода:

1. В окне профиля выберите раздел **PIN-код**.
2. Введите новый PIN-код, состоящий из 6 цифр.
3. Нажмите кнопку "Сохранить".

Примечание: PIN-код используется только для входа в мобильное приложение.

<details>

<summary>Блокировка (Lock)</summary>

При нажатии кнопки **Блокировка** в меню профиля система временно блокируется.

* Экран блокируется.
* Выход из системы не происходит.
* Для повторного входа требуется ввести пароль.

Эта функция обеспечивает безопасность, когда пользователь временно отходит от компьютера.

<details>

<summary>Выход (Logout)</summary>

При нажатии кнопки **Выход** пользователь полностью выходит из системы.

* Сеанс завершается.
* Для повторного входа требуется логин и пароль.
