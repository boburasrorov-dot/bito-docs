### Боковое меню (Навигационная панель)

Боковое меню используется для навигации между модулями системы.

Каждый модуль управляет определенным бизнес-процессом:

* **Отчеты** – просмотр аналитических и статистических данных
* **Продажи** – документы и операции продаж
* **CRM** – работа с клиентами
* **Снабжение** – закупки и поставки
* **Склад** – движение товаров
* **Финансы** – денежный поток и расчеты
* **Производство** – производственный процесс
* **Настройки** – конфигурация системы

Цель: управление бизнес-процессами по модулям

<div><figure><img src="../../.gitbook/assets/image (196).png" alt=""><figcaption></figcaption></figure> <figure><img src="../../.gitbook/assets/image (197).png" alt=""><figcaption></figcaption></figure></div>

У каждого модуля также есть свои подмодули, то есть действия, входящие в этот модуль, например:

* Продажи - раздел для создания новой продажи и просмотра всех продаж
* Этикетки - раздел для печати этикеток
* Возврат - раздел для возврата проданных товаров и просмотра возвращенных товаров
* Рассрочка - раздел для контроля рассрочных платежей
* Планирование - раздел для постановки плана по продажам и его отслеживания
* Дашборд продаж - общий дашборд по продажам
* Платежи клиентов - раздел для управления и мониторинга платежей должников

### Фильтрация

В BITO функция фильтрации помогает быстро и точно находить нужные данные. На каждой странице с таблицей есть фильтр, и пользователь может сортировать данные по дате, статусу, ответственному лицу, складу, клиенту и другим столбцам.

В BITO доступны 3 вида отображения фильтра:

* Встроенный (Inline)
* Статичный
* Сворачиваемый

### Настройка вида фильтра

Пользователь имеет возможность сделать одинаковым вид фильтра на всех страницах:

**Настройки → Настройки пользователя → Фильтр → Выбор**

Здесь пользователь может выбрать удобный для себя тип фильтра (встроенный, статичный или разворачиваемый) и установить его как стандартный вид по всей системе.
