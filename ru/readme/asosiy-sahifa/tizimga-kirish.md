# Вход в систему

<details>

<summary>Выбор языка перед входом в систему</summary>

Перед входом в систему BITO вы можете выбрать язык интерфейса.

1. Откройте страницу входа (например: dokon.bito.uz).
2. Нажмите на кнопку выбора языка, расположенную **в правом верхнем углу** страницы.
3. Выберите необходимый язык.
4. Интерфейс системы автоматически изменится на выбранный язык.

<summary>Вход по номеру телефона и паролю</summary>

Для входа в систему требуется:

* Ваш номер телефона
* Пароль, выданный администратором организации

<figure><img src="../../.gitbook/assets/image (159).png" alt=""><figcaption></figcaption></figure>

#### Порядок входа:

1. Введите адрес системы \
   Например: dokon.bito.uz \
   (Для каждой организации будет отдельный адрес.)
2. Введите ваш номер телефона
3. Введите пароль
4. Если кнопка "Запомнить меня" нажата, при входе в систему в течение 30 дней будет выполняться автоматический вход. Через 30 дней потребуется снова ввести логин и пароль.
5. Нажмите кнопку "Войти"

Если данные верны, вы успешно войдете в систему.

<summary>Восстановление пароля</summary>

Если вы забыли пароль, можете самостоятельно восстановить его.

1. Откройте страницу входа в систему (например: dokon.bito.uz).
2. Нажмите кнопку **"Забыли пароль?"**.
3. Введите ваш номер телефона.
4. На ваш номер будет отправлен код подтверждения по SMS.
5. Введите SMS-код.
6. Установите новый пароль.
7. Войдите в систему с новым паролем.

<summary>Вход через одноразовый пароль (SMS)</summary>

В системе BITO пользователь может войти в систему также через **одноразовый SMS-код**, не вводя пароль.

* Перейдите в окно "**Одноразовый пароль**"
* Введите зарегистрированный в системе **номер телефона**.
* Нажмите на **значок SMS (иконку)** рядом с полем номера телефона.
* На введенный номер будет отправлен одноразовый код подтверждения по SMS.
* Введите полученный SMS-код в специальное поле.
* Нажмите кнопку **"Войти"**.
* Если код верен, вы успешно войдете в систему.
