# Ожидаемые товары





### Работа с ожидаемыми товарами

При входе в **Маркетинг → Ожидаемые товары** отображается список всех ожидаемых товаров, созданных в системе.

Через этот раздел можно заранее внести продукцию, поступление которой ожидается на склад в будущем. Это в основном работает на основе заказа клиента.

Например, если клиент запрашивает определенный продукт, которого в настоящее время нет в наличии на складе, через этот раздел вносится товар и его количество, а также привязывается клиент. Впоследствии, когда этот товар поступит на склад, можно уведомить клиента через систему.

Если система BITO интегрирована с Eskiz.uz или другими SMS-платформами, при поступлении товара на склад клиенту автоматически отправляется SMS-уведомление.

Для добавления нового ожидаемого товара нажимается кнопка Создать и вводятся следующие данные:

• **Продукт** — выбирается ожидаемый товар.

• **Количество** — вносится количество данного товара, которое ожидается.

• **Клиент** — выбирается клиент, ожидающий товар.

• **Комментарий** — можно внести дополнительную информацию или примечание.

• **Загрузить файл** — прикрепляются документы, связанные с товаром или заказом.

После ввода всех данных они сохраняются и берутся под контроль в системе.

В результате создается возможность уведомления клиента при поступлении товара на склад. Это помогает улучшить работу с клиентами и не упустить возможности продаж.
