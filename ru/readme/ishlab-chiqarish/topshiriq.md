# Задание



### Создание задания

Раздел "Задание" используется для практического запуска и управления производственным процессом. Здесь на основе ранее созданных тех. карт создаются производственные задачи (задания) и контролируется рабочий процесс по ним.

Переход в раздел Производство → **Задание** и создание нового задания.

<figure><img src="../../.gitbook/assets/image (28).png" alt=""><figcaption></figcaption></figure>

При создании задания вводятся следующие основные данные:

* **Ответственное лицо**\* – сотрудник, ответственный за задание
* **Производственный склад**\* – склад, на котором осуществляется процесс
* **Валюта**\* – валюта для расчетов
* **Код** – идентификатор задания
* **Примечание** – дополнительная информация



Самая важная часть задания — здесь **выбирается тех. карта**.

* При выборе **тех. карты** система автоматически **загружает связанные с ней этапы**
* Каждый этап отражается как последовательный шаг в производственном процессе
* Вводятся **продукт** и **количество**
* **Норма** – стандартный производственный показатель
* **Запланировано** – объем, планируемый к выполнению
