# Цены

<details open>

<summary>Создание цен</summary>

Раздел **Цены** позволяет создавать различные типы цен для товаров и управлять ими централизованно. Через этот раздел пользователь может устанавливать цены, соответствующие различным торговым стратегиям, применять их к товарам и контролировать общую ценовую политику. Кроме того, имеется возможность автоматического расчета цен и управления через ограничения (минимальные/максимальные). Этот раздел используется для ведения ценовой политики и повышения эффективности продаж.

<figure><img src="../../.gitbook/assets/image (2).png" alt=""><figcaption></figcaption></figure>

Осуществляется вход в раздел Данные → Цены, и через кнопку **"Создать"** создается новый тип цены.

В этом разделе все цены отображаются в виде таблицы (table).

Столбцы таблицы:

* **Название** – название типа цены (например: розничная, оптовая)
* **Код** – краткий код для цены
* **Тип** – тип цены (Продажа или Закуп)
* **Создавший** – пользователь, создавший цену
* **Дата создания** – дата и время создания цены
* **Изменивший** – пользователь, внесший последнее изменение
* **Дата изменения** – дата и время последнего обновления
* **Действия** – кнопка "Установить цены" для применения цены к товарам

<figure><img src="../../.gitbook/assets/image (4).png" alt=""><figcaption></figcaption></figure>

Созданные цены присваиваются товарам через кнопку **"Установить цены"** справа. В этом окне цены по товарам могут вноситься вручную или автоматически (на основе процента или суммы).

Этот раздел служит для управления общими ценами, в то время как внутри товара отображается только цена, относящаяся к этому товару. По этой причине пользователь может управлять ценами по всем товарам не по отдельности, а в одном месте. Это дает возможность быстро обновлять цены, сокращать ошибки и оптимизировать процесс продаж.
