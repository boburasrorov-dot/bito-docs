# Ежедневные визиты



### Ежедневные визиты

Раздел ежедневных визитов позволяет просматривать визиты, запланированные агентами на выбранную дату. Через этот раздел пользователь может просмотреть, какой агент должен посетить каких клиентов в определенный день, контролировать их и эффективно управлять ежедневным рабочим процессом. Этот раздел является важным инструментом для оперативного контроля и ежедневного планирования.

Осуществляется вход в раздел Визиты → Ежедневные визиты. В верхней части имеется фильтр по дате, с помощью которого пользователь может выбрать нужную дату и просмотреть визиты, относящиеся именно к этому дню.

В этом разделе все визиты отображаются в виде таблицы (table).

Столбцы таблицы:

* **№** – порядковый номер
* **Название** – имя агента (сотрудника)
* **Количество клиентов** – количество клиентов, которых необходимо посетить в этот день
* **Заказ** – заказы, связанные с визитом (если имеются)
* **Действия** – операции (например, "Поставить план")

При нажатии на каждого агента открывается его подробная информация. В этом окне отображаются клиенты, прикрепленные к агенту, и статус их посещения.

Столбцы таблицы в подробном окне:

* **№** – порядковый номер
* **Название** – название клиента
* **Тип** – тип клиента (например, физическое лицо)
* **План** – тип визита (например, автоматический)
* **Очередь** – последовательность или статус визита

Основная особенность этого раздела заключается в том, что он работает только в рамках одной даты и показывает визиты, которые должны быть выполнены именно в этот день. Это дает пользователю возможность просматривать только важные ежедневные задачи без лишней информации.

В результате усиливается контроль над деятельностью агентов, визиты выполняются вовремя, а процесс работы с клиентами становится более систематизированным и эффективным.
