# Тип видеоотчета



### Тип видеоотчета

Раздел "Тип видеоотчета" позволяет создавать и управлять типами (категориями) видеоотчетов, предоставляемых агентами. Через этот раздел пользователь заранее определяет, какие типы видео требуются от агентов. Например, создаются такие типы, как "Видео интерьера магазина" или "Видео размещения товара", и от агента требуется загружать именно эти типы видео в процессе визита. Этот раздел используется для усиления контроля и повышения качества визитов.

Для входа в раздел используется путь: Визиты → Тип видеоотчета. В этом разделе все созданные типы видеоотчетов отображаются в виде таблицы.

Столбцы таблицы:

* **№** – порядковый номер
* **Полное название** – название типа видеоотчета

Для создания нового типа видеоотчета нажимается кнопка "Создать". В открывшемся окне присутствуют следующие поля:

* **Название** – название типа видеоотчета (например, "Интерьер магазина", "Видеоотчет")
* **Обязательно** – определяет, является ли загрузка видео данного типа обязательной или нет
* **Активно** – определяет, будет ли тип находиться в активном состоянии

Также присутствуют следующие управляющие кнопки:

* **Отменить** – отменить действие
* **Сохранить** – сохранить данные

<figure><img src="../../.gitbook/assets/image (507).png" alt=""><figcaption></figcaption></figure>

Основная особенность данного раздела заключается в том, что он работает аналогично типам фотоотчетов и устанавливает четкие требования для агентов. Если создано несколько типов видеоотчетов, агенту потребуется загружать видео по всем этим типам во время визита (если установлена отметка "Обязательно").

В результате компания организует полный контроль через видео в процессе визитов, создается возможность более точной оценки деятельности агентов и повышается качество отчетности.
