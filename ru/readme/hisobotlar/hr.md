# HR



### Продажи по сотрудникам

<figure><img src="../../.gitbook/assets/image (245).png" alt=""><figcaption></figcaption></figure>

Данный отчет отображает показатели продаж, осуществленных сотрудниками. С помощью отчета можно проанализировать объем продаж каждого сотрудника, возвращенные товары, скидки, налоги, а также итоговый доход. Этот отчет помогает оценить процесс продаж в разрезе сотрудников.

<figure><img src="../../.gitbook/assets/image (244).png" alt=""><figcaption></figcaption></figure>

**В таблице отчета представлены следующие столбцы:**

**№** — порядковый номер в таблице.

**Наименование** — имя сотрудника, осуществившего продажу.

**Продано** — количество продаж, совершенных сотрудником.

**Проданное количество** — общее количество проданных товаров.

**Возвращенное количество** — количество товаров, возвращенных клиентами.

**Продажи** — общая сумма продаж проданных товаров.

**Возвращено** — общая стоимость возвращенных товаров.

**Скидка** — сумма скидок, примененных в процессе продаж.

**Входящий налог** — сумма входящего налога, рассчитанного в процессе продаж.

**Добавленный налог** — сумма добавленного налога, рассчитанного в процессе продаж.

**Доход** — чистый доход, полученный в результате продаж.

**Себестоимость** — общая себестоимость проданных товаров.

**Средний чек** — средняя сумма дохода на одну продажу.

**Количество новых клиентов** — количество новых клиентов, привлеченных сотрудником.

**Валовой доход** — общая прибыль после учета себестоимости.



**Польза для бизнеса**

С помощью данного отчета можно:

* анализировать эффективность продаж по сотрудникам
* сравнивать объем продаж и доходы в разрезе сотрудников
* контролировать возвращенные товары и скидки
* оценивать результативность сотрудников в процессе продаж
* определять эффективность привлечения новых клиентов
