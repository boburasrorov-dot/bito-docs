# Дистрибуция



### Отчет по визитам

<figure><img src="../../.gitbook/assets/image (246).png" alt=""><figcaption></figcaption></figure>

Данный отчет отображает общую информацию о визитах, совершенных торговыми агентами. Отчет позволяет анализировать запланированные визиты агентов, совершенные визиты, заказы, результаты продаж, а также степень успешности визитов.

**В отчете доступны следующие основные показатели:**

**Визиты (по визитам)** — общий процент визитов, совершенных агентами.

**Успешные визиты** — процент визитов, завершившихся заказом или продажей.

**Визиты (через GPS)** — процент визитов, подтвержденных через GPS.

**Фото отчеты** — доля фото отчетов, загруженных во время визита.

**Общая сумма** — сумма продаж, совершенных в выбранном периоде.

**Совершенные продажи** — количество продаж, совершенных во время визитов.

**В таблице отчета доступны следующие столбцы:**

**№** — порядковый номер в таблице.

**Сотрудник** — агент или торговый сотрудник, совершивший визиты.

**Запланировано** — количество запланированных визитов для агента.

**Визит** — количество совершенных агентом визитов.

**Без визита** — количество визитов, которые были запланированы, но не совершены.

**Фото отчет** — количество фото отчетов, загруженных во время визита.

**Продажа** — сумма продажи, совершенной во время визита.

**Количество** — общее количество проданной продукции.

**Неуспешные** — количество визитов, не завершившихся заказом или продажей.

**Внеплановый визит** — количество незапланированных, но совершенных визитов.



**Польза для бизнеса**

С помощью данного отчета можно:

* контролировать визитную активность торговых агентов
* оценивать эффективность работы агентов
* сравнивать запланированные и совершенные визиты
* анализировать продажи, совершенные через визиты
* эффективно управлять деятельностью торговых агентов
