<summary>Остаток товаров по организациям</summary>

Данный отчет показывает остатки товаров в разрезе организаций. С помощью отчета можно увидеть, сколько каждого товара имеется в различных организациях или филиалах. Этот отчет позволяет анализировать остатки товаров на складах или в филиалах в одном месте.

<figure><img src="../../.gitbook/assets/image (261).png" alt=""><figcaption></figcaption></figure>

**№** — порядковый номер в таблице.

**Изображение** — фото товара.

**Товар** — наименование товара.

**Единица измерения** — единица измерения товара (например: штука, килограмм, литр).

**Количество** — общее количество остатков товара по всем организациям.

**Названия организаций (например: Jarvis, My company, My do'kon и другие)** — количество остатков товара в каждой организации или филиале.

**Итого** — в конце таблицы показано общее количество остатков по всем товарам.



**Польза для бизнеса**

С помощью данного отчета:

* можно просматривать остатки товаров по различным филиалам или организациям
* можно быстро определить, какой товар есть в каком филиале
* можно анализировать распределение товаров между складами
* можно выявлять избыточные или недостающие товары
* можно оптимизировать логистику и управление складом

<summary>Отчет по перемещениям</summary>

<figure><img src="../../.gitbook/assets/image (250).png" alt=""><figcaption></figcaption></figure>

Данный отчет показывает операции перемещения товаров, выполненные в системе. С помощью отчета можно увидеть, из какой организации или склада куда были перемещены товары, причину перемещения, количество и статус процесса.

Этот отчет используется для контроля движения товаров между складами или внутри одной организации.

<figure><img src="../../.gitbook/assets/image (260).png" alt=""><figcaption></figcaption></figure>

**В таблице отчета присутствуют следующие колонки**

**№** — порядковый номер операции перемещения.

**Причина** — причина перемещения товара.

**Номер** — номер документа перемещения.

**Код** — код операции перемещения в системе.

**Дата** — дата и время создания операции перемещения.

**Тип** — тип перемещения. Например:

* Внутреннее перемещение
* Взаимное перемещение

**Организация-отправитель** — организация, отправившая товар.

**Организация-получатель** — организация, принимающая товар.

**Склад-отправитель** — склад, с которого отправлен товар.

**Склад-получатель** — склад, принимающий товар.

**Статус** — статус операции перемещения. Например:

* В процессе
* Выполнено

**Общее количество** — общее количество товара, указанное для перемещения.

**Завершенное количество** — количество успешно перемещенного товара.

**Отмененное количество** — количество отмененного товара.

**Количество в процессе** — количество товара, находящегося в процессе перемещения.

**Количество типов товаров** — количество типов товаров в документе перемещения.

**Общая себестоимость** — общая себестоимость перемещаемых товаров.



**С помощью данного отчета:**

* можно контролировать движение товаров между складами
* можно увидеть, из какой организации куда был отправлен товар
* можно отслеживать статус процесса перемещения
* можно анализировать причины перемещения товаров
* можно эффективно организовать логистику и управление складом

<summary>Приход-расход товаров</summary>

<figure><img src="../../.gitbook/assets/image (255).png" alt=""><figcaption></figcaption></figure>

Данный отчет показывает движение прихода и расхода товаров по выбранному периоду времени и складу. С помощью отчета можно детально проанализировать, откуда товары поступили на склад и через какие операции ушли со склада.

Отчет формируется после выбора **даты** и **склада**.

<figure><img src="../../.gitbook/assets/image (263).png" alt=""><figcaption></figcaption></figure>

**№** — порядковый номер в таблице.

**Товары** — наименование товара.

**Единица измерения** — единица измерения товара (например: килограмм, штука, литр).

**Категория** — категория, к которой относится товар.

**Начальное количество** — количество товара, имевшееся на складе в начале выбранного периода.



**Раздел прихода**

Эта часть показывает, через какие операции товар поступил на склад.

**Приход** — количество товара, поступившего на склад.

**Инвентаризация** — количество товара, добавленного в процессе инвентаризации.

**Возврат продаж** — количество товара, возвращенного клиентом.

**Внутреннее перемещение (приход)** — товар, поступивший в результате внутреннего перемещения с другого склада.

**Взаимное перемещение (приход)** — товар, поступивший в результате перемещения из другой организации.

**Произведено** — количество товара, добавленного на склад в результате производства.



**Раздел расхода**

Эта часть показывает, через какие операции товар ушел со склада.

**Продажи** — количество проданного товара.

**Возврат закупок** — количество товара, возвращенного поставщику.

**Инвентаризация (расход)** — количество товара, уменьшенного в процессе инвентаризации.

**Утилизация** — количество товара, признанного непригодным и списанного.

**Внутреннее перемещение (расход)** — товар, отправленный на другой склад в результате внутреннего перемещения.

**Взаимное перемещение (расход)** — товар, отправленный в другую организацию в результате перемещения.



**Польза для бизнеса**

С помощью данного отчета:

* можно отслеживать движение товаров на склад и со склада
* можно увидеть, какие операции повлияли на остатки товаров
* можно детально анализировать складские операции
* можно контролировать результаты инвентаризации
