# Заказ на закупку



### Создание заказа на закупку

В разделе Снабжение → Заказ на закупку при нажатии кнопки **Создать** открывается окно нового заказа на закупку.

Через данный раздел на основе потребностей предприятия формируются официальные заказы поставщикам и планируются процессы закупок. Заказ на закупку позволяет заранее заказывать продукцию, согласовывать цены и системно управлять процессом снабжения. Также через данный раздел контролируются будущие закупки и все заказы ведутся в единой системе.

При создании заказа на закупку вводятся следующие данные:

• **PO номер** — вводится номер заказа \
• **Статус** — выбирается статус заказа.\
• **Дата** — указывается дата создания заказа.\
• **Дата закупки** — вводится планируемая дата закупки.\
• **Ответственное лицо** — назначается ответственный сотрудник за заказ.\
• **Поставщик** — выбирается поставщик продукции.\
• **Валюта** — выбирается валюта.\
• **Загрузить файл** — прикрепляются документы.\
• **Комментарий** — вводится дополнительная информация.

Через раздел заявок на закупку существующие заявки прикрепляются к заказу.

Через раздел дополнительных продуктов и сырья:

• **Наименование** — выбирается продукт.\
• **Единица измерения** — указывается единица измерения.\
• **Количество** — вводится количество заказа.\
• **Количество коробок** — вводится количество упаковок.\
• **Общее количество** — рассчитывается общий объем.\
• **Цена** — вводится цена продукта.\
• **Общая цена** — рассчитывается общая стоимость.

<figure><img src="../../.gitbook/assets/image (70).png" alt=""><figcaption></figcaption></figure>

После ввода всех данных сохраняется и создается заказ на закупку.\
В результате формируется официальный заказ для отправки поставщику и процесс закупки управляется по четкому плану.
