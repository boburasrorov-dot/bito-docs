# Возврат



### Возвраты

Через раздел Снабжение → Возврат осуществляется процесс возврата товаров по закупкам, выполненным в системе. В данном разделе возврат производится простым и оперативным способом — все данные вводятся в одном окне без разделения на отдельные разделы.

<figure><img src="../../.gitbook/assets/image (78).png" alt=""><figcaption></figcaption></figure>

Процесс возврата осуществляется двумя способами:

#### **Создание через раздел Возврат**

Пользователь заходит в раздел Снабжение → Возврат и создает новый документ возврата.

В данном окне вводятся следующие данные:

• **Склад** — выбирается склад, на который будут приняты возвращенные товары.\
• **Закупка** — выбирается документ закупки, на основании которого осуществляется возврат (опционально).\
• **Поставщик** — выбирается поставщик, которому возвращается товар.\
• **Номер SRN** — дополнительный номер, относящийся к документу возврата (опционально).\
• **Время** — указывается дата и время осуществления возврата.\
• **Причина** — выбирается причина возврата (обязательно).\
• **Ответственное лицо** — указывается сотрудник, ответственный за возврат.\
• **Валюта** — выбирается валюта, в которой осуществляется расчет возврата.\
• **Загрузить файл** — прикрепляются необходимые документы.\
• **Примечание** — записывается дополнительная информация.

<figure><img src="../../.gitbook/assets/image (80).png" alt=""><figcaption></figcaption></figure>

После этого в нижней части пользователь выбирает возвращаемые товары:\
• добавляются необходимые товары\
• вводится возвращаемое количество

Система автоматически рассчитывает общую сумму на основании введенных данных.

После заполнения всех данных документ сохраняется и возврат осуществляется.

#### **Возврат из документа закупки**

Второй способ возврата — осуществляется из существующего документа закупки.

Для этого пользователь:

* открывает необходимый документ закупки
* нажимает кнопку **"Возврат"**, расположенную справа

<figure><img src="../../.gitbook/assets/image (81).png" alt=""><figcaption></figcaption></figure>

В результате открывается окно возврата, и некоторые данные заполняются автоматически (поставщик, товары и прочие).

Пользователь:\
• выбирает причину возврата\
• вводит количество необходимых товаров\
• при необходимости вводит дополнительные данные

<figure><img src="../../.gitbook/assets/image (82).png" alt=""><figcaption></figcaption></figure>

После этого документ возврата сохраняется.

В результате процесс возврата осуществляется просто и быстро, товары возвращаются на склад и все операции полностью отражаются в системе.
