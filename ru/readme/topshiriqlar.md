# Задачи

<details open>

<summary>Работа с задачами</summary>

При входе в **CRM → Задачи** отображается список всех задач, созданных в системе. Доступна возможность просмотра задач в разрезе дня, недели и месяца.

Через данный раздел можно назначать задачи сотрудникам, контролировать их и отслеживать статус выполнения.

Задачи могут быть привязаны к продажам, работе с клиентами или внутренним процессам. Это помогает организованно вести дела.

<figure><img src="../.gitbook/assets/image (314).png" alt=""><figcaption></figcaption></figure>

Для создания новой задачи нажимается кнопка **Создать** и вводятся следующие данные:

<figure><img src="../.gitbook/assets/image (315).png" alt=""><figcaption></figcaption></figure>

• **Продажа** — выбирается, с какой продажей или процессом связана задача.

• **Ответственное лицо** — назначается исполнитель задачи.

• **Тип** — тип задачи (например, долг, задание и другие).

• **Дата и время** — указывается временной интервал, в который должна быть выполнена задача.

• **Примечание** — записывается дополнительная информация по задаче.

Задачи отслеживаются в виде таблицы или календаря.

Можно контролировать задачи в зависимости от статуса (например: выполнена, не выполнена, просрочена).

В результате через задачи контролируется активность сотрудников и организованно управляются рабочие процессы.
