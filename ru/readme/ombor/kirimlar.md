# Приходы

<details open>

<summary>Приходы</summary>

Раздел Приходы предназначен для осуществления процесса приемки товаров на склад. Данный процесс обычно выполняется на основе ранее созданных и завершенных документов закупки.

При создании прихода пользователь выбирает **завершенную закупку, в которой не включена прямая приемка**. После выбора закупки список товаров, относящихся к данной закупке, автоматически формируется и загружается в окно прихода.

Пользователь переходит в раздел **Склад → Приходы** и нажимает кнопку **"Создать"**.

<figure><img src="../../.gitbook/assets/image (42).png" alt=""><figcaption></figcaption></figure>

#### **Основные данные**

На этом этапе вносятся общие данные документа прихода:

* Номер прихода
* Дата прихода
* Ответственное лицо
* Поставщик
* Склад
* Валюта
* Примечание и файл (необязательно)

<figure><img src="../../.gitbook/assets/image (43).png" alt=""><figcaption></figcaption></figure>

#### **Товары**

Здесь выбирается **Закупка**, и автоматически отображаются товары, относящиеся к выбранной закупке.

Пользователь:

* проверяет количество
* при необходимости изменяет
* может провести частичный приход

<figure><img src="../../.gitbook/assets/image (44).png" alt=""><figcaption></figcaption></figure>

#### **Дополнительные расходы**

Транспортные, таможенные и прочие дополнительные расходы вносятся здесь.\
Эти расходы добавляются к себестоимости товара.
