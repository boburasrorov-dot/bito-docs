# Причины списания



### Причины списания

Через раздел причин списания ведется список причин, используемых при списании товаров со склада. Эти причины впоследствии выбираются в процессе списания и точно показывают, по какой причине товар был списан со склада.

<figure><img src="../../.gitbook/assets/image (367).png" alt=""><figcaption></figcaption></figure>

Пользователь переходит в раздел **Склад → Причины списания** и нажимает кнопку **"Создать"** для добавления новой причины.

В этом разделе создаются и управляются стандартные причины для списания товаров.

Например:

* Брак (дефектный товар)
* Потеря
* Истечение срока годности
* Повреждение

При создании новой причины вводятся следующие данные:

Наименование — название причины списания\
Код — код внутренней идентификации (необязательно)\
Порядковый номер — определяет порядок отображения в списке\
Примечание — дополнительная информация (необязательно)\
Активность — причина может быть в активном или неактивном состоянии
