# Списание

<details open>

<summary>Списание </summary>

Раздел списаний позволяет осуществлять процесс списания товаров со склада по различным причинам. Этот процесс используется для оформления вывода товаров со склада в связи с приходом в негодность, утерей или другими причинами.

Пользователь переходит в раздел **Склад → Списания** и нажимает кнопку **"Создать"** для создания нового документа.

<figure><img src="../../.gitbook/assets/image (369).png" alt=""><figcaption></figcaption></figure>

При создании списания вводятся следующие данные:

* **WO номер** — номер документа
* **Дата** — дата списания
* **Склад** — склад, с которого списывается товар
* **Ответственное лицо** — ответственный сотрудник
* **Причина** — предварительно созданная причина списания
* **Валюта** — валюта расчета
* **Действие** — тип списания\
  (например: пометить как брак и другие)
* **Склад брака** — если товар выделяется как брак, на какой склад он переводится
* **Комментарий** — дополнительная информация (необязательно)
* **Загрузить файл** — прикрепление документа (необязательно)

В нижней части выбираются товары для списания.

Пользователь:

* ищет или выбирает товары
* вводит списываемое количество
* видит доступное количество на складе

<figure><img src="../../.gitbook/assets/image (370).png" alt=""><figcaption></figcaption></figure>

По каждому товару:

* **Количество на складе** — текущий остаток
* **Количество** — списываемое количество
* **Количество в коробках** — количество по упаковке
* **Срок годности** — если имеется
