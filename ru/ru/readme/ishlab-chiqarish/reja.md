# План

<details>

<summary>Создание плана</summary>

Раздел План используется для предварительного планирования производства и определения объемов производства продукции в разрезе месяцев в течение года. Через этот раздел определяется, сколько продукции должно быть произведено по каждому месяцу, и впоследствии сравнивается с фактическими результатами.

Необходимо войти в раздел Производство → **План** и через кнопку **"Создать"** создать новый план.

<figure><img src="../../.gitbook/assets/image (21).png" alt=""><figcaption></figcaption></figure>

При создании плана вводятся следующие основные данные:

* **Год**\* – год составления плана (например: 2026)
* **Примечание** – дополнительная информация или примечание

В плане добавляются продукты и для каждого из них вводится количество в разрезе месяцев:

* **Продукт** – выбирается производимая продукция
* **Январь – Декабрь** – плановое количество производства для каждого месяца

Здесь пользователь полностью распределяет объем производства в течение года.

<figure><img src="../../.gitbook/assets/image (29).png" alt=""><figcaption></figcaption></figure>

При входе в план отображаются следующие данные:

* **План** – установленное количество
* **Факт** – фактически произведенное количество
* **Выполнено (%)** – степень выполнения плана

Эти данные отражаются в разрезе каждого месяца и в общем (итого) виде.

<figure><img src="../../.gitbook/assets/image (30).png" alt=""><figcaption></figcaption></figure>

</details>
