# Визиты

<details open>

<summary>Визиты</summary>

Раздел "Визиты" позволяет просматривать все завершенные визиты, выполненные в системе, в едином месте. Через этот раздел пользователь может просматривать историю визитов, выполненных агентами, анализировать результаты продаж, созданные на их основе, и оценивать общую эффективность деятельности. Этот раздел имеет важное значение для процессов отчетности и анализа.

<figure><img src="../../.gitbook/assets/image (502).png" alt=""><figcaption></figcaption></figure>

Осуществляется вход в раздел **Визиты → Визиты**. В этом разделе все визиты отображаются в виде таблицы (table), и пользователь имеет возможность их поиска, фильтрации по дате или сотруднику.

Столбцы таблицы:

* **№** – порядковый номер
* **Номер** – уникальный идентификатор визита
* **Организация** – организация (филиал или компания), в которой был выполнен визит
* **Дата** – дата выполнения визита
* **Общая цена** – сумма продажи, осуществленной во время визита
* **Клиент** – название клиента, которому был нанесен визит
* **Агент** – сотрудник, выполнивший визит

В этом разделе также доступны дополнительные возможности:

* **Поиск** – для быстрого поиска визитов
* **Фильтр (Дата, Сотрудник)** – для сортировки данных по необходимым параметрам
* **Экспорт** – для загрузки списка всех визитов

Основная особенность этого раздела заключается в том, что он показывает только завершенные визиты. То есть здесь сохраняются выполненные и зарегистрированные в системе визиты. Это предоставляет пользователю точные данные для анализа прошедшей деятельности, оценки эффективности агентов и планирования на будущее.
