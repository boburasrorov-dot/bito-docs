# Video Report Type



### Video Report Type

The Video Report Type section allows creating and managing types (categories) for video reports submitted by agents. Through this section, the user predefines what types of videos are required from agents. For example, types such as "Store Interior Video" or "Product Placement Video" are created, and the agent is required to upload exactly these types of videos during the visit. This section is used to strengthen control and improve visit quality.

Navigate to Visits → Video Report Type section. In this section, all created video report types are displayed in table view.

Table columns:

* **№** – sequence number
* **Full Name** – video report type name

To create a new video report type, click the "Create" button. In the opened window, the following fields are available:

* **Name** – video report type name (for example, "Store Interior", "Video Report")
* **Mandatory** – specifies whether uploading this type of video is mandatory or not
* **Active** – specifies that the type is in active status

Additionally, the following control buttons are available:

* **Cancel** – cancel the action
* **Save** – save the data

<figure><img src="../../.gitbook/assets/image (507).png" alt=""><figcaption></figcaption></figure>

The main feature of this section is that it works similarly to photo report types and sets clear requirements for agents. If multiple video report types have been created, the agent will need to upload videos for all these types during the visit (if "Mandatory" is specified).

As a result, the company establishes full control through video during visits, creates the ability to evaluate agent activities more accurately, and improves report quality.
