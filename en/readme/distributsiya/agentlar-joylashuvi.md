# Agent Locations



### Viewing Agent Locations

The Agent Locations section provides the ability to track agents' geographic locations via map in near real-time. Through this section, users can visually monitor where agents are located, their activity status, and the progress of visit execution. This section serves as an important tool for logistics, control, and operational management.

Access is made through **Visits → Agent** Locations section. In this window, the main area displays a map showing agent locations. On the right side, there is a monitoring panel.

The monitoring panel provides date selection capability, allowing users to specify which day's data to view. Additionally, the following general indicators are displayed:

* **All agents** – total number of agents in the system
* **Active agents** – number of currently active agents

This section contains three main tabs: Agents, Visits, and Reports.

The Agents tab displays the list of agents.

Table columns:

* **Employee** – agent (employee) name
* **Amount** – sales amount executed by the agent
* **Status** – agent's current status (e.g., not started work)

The Visits tab displays visits executed by agents.

Table columns:

* **Customer** – name of the customer visited
* **Amount** – sales amount executed during the visit
* **Time** – time when the visit was executed

The Reports tab displays general statistical indicators:

* **By plan** – number of planned visits
* **Visited** – number of executed visits
* **Remaining** – number of unexecuted visits
* **Average time** – average time spent on visits
* **Successful visits** – visits completed with results
* **Number of orders** – orders created based on visits
* **Total price** – total sales amount
* **Average price** – average check amount
* **Average quantity of goods** – average number of products sold

The main feature of this section is that it presents agent activity in visual form through the map and simultaneously provides near real-time monitoring capability. This enables managers to control agent movements, assess efficiency, and make prompt decisions.
