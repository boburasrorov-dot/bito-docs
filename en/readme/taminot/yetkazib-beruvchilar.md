# Suppliers



### Working with suppliers

In the Supply → Suppliers section, clicking the **Create** button opens a new supplier creation window.

<figure><img src="../../.gitbook/assets/image (62).png" alt=""><figcaption></figcaption></figure>

Through this section, all suppliers working with the enterprise are entered into the system and information related to them is stored in a unified database. Proper supplier management simplifies purchasing processes and enables accurate management of contracts and settlements.

When creating a supplier, the following information is entered:

<figure><img src="../../.gitbook/assets/image (46).png" alt=""><figcaption></figcaption></figure>

<figure><img src="../../.gitbook/assets/image (47).png" alt=""><figcaption></figcaption></figure>

• **Type** — supplier type is selected (individual or legal entity).\
• **Legal entity** — if available, legal entity is selected.\
• **Company name** — supplier name is entered.\
• **TIN** — tax identification number is entered.\
• **Address** — supplier address is entered.\
• **Phone** — phone number is entered.\
• **Responsible person** — employee working with this supplier is designated.\
• **Category** — supplier category is selected.\
• **Upload file** — documents related to the supplier are attached.\
• **Notes** — additional information or notes are written.

Additional information:

• **Image** — company logo or image is uploaded.\
• **Short name** — abbreviated name is entered.\
• **Code** — internal identification code is entered.\
• **Postal code** — postal code is entered.\
• **Parent company** — related company is selected.\
• **Manager** — manager working with the supplier is designated.\
• **Telegram** — Telegram account is entered.\
• **Email** — email address is entered.\
• **Website** — website address is entered.\
• **Location** — location can be marked on the map.\
• **Status** — supplier is marked as active or inactive.\
• **Customer** — if the supplier is also a customer at the same time, this flag is enabled.

After all information is entered, it is saved and the supplier is added to the system.\
As a result, the process of working with suppliers is centralized, creating the ability to quickly select and control during purchasing processes.
