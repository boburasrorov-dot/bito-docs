# Label

<details open>

<summary>Label printing</summary>

The Label function is used to prepare and print **labels (tags)** for products. The appearance of the label and what information is displayed on it is configured in **settings**.

<figure><img src="../../.gitbook/assets/image (297).png" alt=""><figcaption></figcaption></figure>

The user can enter the **Sales → Label** section to prepare and print labels for products.

When creating a label, first a **label type (template)** is selected. Each template defines the size and appearance of the label.

<figure><img src="../../.gitbook/assets/image (296).png" alt=""><figcaption></figcaption></figure>

Then the required product is selected from the product list.

Additionally, it can be configured whether the price is displayed or not.

After products are added, the system **prepares the number of labels based on the entered quantity**. The user can print the labels using the **Print** button.

This allows you to quickly and conveniently prepare and print the required number of labels for products.
