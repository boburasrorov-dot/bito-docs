# Export Center

<details open>

<summary>Export Center</summary>

The Export Center section allows you to view all export processes performed in the system in a single location. Through this section, users can monitor what data has been exported, by whom, and when it was performed.

Access via Tools → Export Center.

All exports are displayed in this section in table view.

<figure><img src="../../.gitbook/assets/image (491).png" alt=""><figcaption></figcaption></figure>

#### Table columns:

* **File name** – name of the exported file
* **User** – user who performed the export
* **Module** – section from which export was performed (for example: Sales)
* **Start time** – export start time
* **Completion date** – export completion time
* **Progress** – export process percentage
* **Status** – export status (Completed, In Progress, etc.)
* **Actions** – file download or delete operations

This section serves as a convenient tool for monitoring export processes, maintaining history, and re-downloading necessary files.

</details>
