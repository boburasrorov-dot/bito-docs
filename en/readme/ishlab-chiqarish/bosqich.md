# Stage



### Creating a Stage

The Stages section allows you to divide the production process into separate steps and manage them in the system. Each stage represents a specific part of production (for example: preparation, processing, packaging, etc.). Through this section, you can organize the production process in a structured and controlled manner.

Go to Production → **Stage** section and create a new stage using the **"Create"** button.

When creating a stage, the following fields are filled in:

* **Name**\* – stage name (for example: Packaging, Mixing)
* **Code** – unique code for the stage
* **Employees** – employees working at this stage
* **Product warehouse** – warehouse where the product will be placed at the end of the stage
* **Note** – additional information

Through this section, you can achieve the following:

* Divide production into stages
* Assign responsible employees for each stage
* Monitor and optimize the process
* Reuse stages in bills of materials
