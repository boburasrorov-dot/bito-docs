# Plan

<details>

<summary>Creating a Plan</summary>

The Plan section is used for advance production planning and setting production volumes by product over the year. Through this section, you can determine how much product should be produced each month and later compare it with actual results.

Navigate to Production → **Plan** section and create a new plan using the **"Create"** button.

<figure><img src="../../.gitbook/assets/image (21).png" alt=""><figcaption></figcaption></figure>

When creating a plan, the following main information is entered:

* **Year**\* – the year for which the plan is being created (example: 2026)
* **Note** – additional information or comment

Products are added within the plan, and quantities are entered by month for each:

* **Product** – the product to be manufactured is selected
* **January – December** – planned production quantity for each month

Here the user fully distributes the production volume throughout the year.

<figure><img src="../../.gitbook/assets/image (29).png" alt=""><figcaption></figcaption></figure>

When entering the plan, the following information is displayed:

* **Plan** – the set quantity
* **Actual** – the actual quantity produced
* **Completed (%)** – the plan completion rate

This information is displayed for each month and in summary (total) view.

<figure><img src="../../.gitbook/assets/image (30).png" alt=""><figcaption></figcaption></figure>

</details>
