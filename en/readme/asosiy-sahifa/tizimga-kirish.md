# System Login



### Selecting a language before logging in

Before logging into the BITO system, you can select the interface language.

1. Open the login page (for example: dokon.bito.uz).
2. Click the language selection button located in the **upper right corner** of the page.
3. Select the desired language.
4. The system interface will automatically switch to the selected language.

### Login via phone number and password

To log into the system, the following are required:

* Your phone number
* Password provided by the organization administrator

<figure><img src="../../.gitbook/assets/image (159).png" alt=""><figcaption></figcaption></figure>

#### Login procedure:

1. Enter the system address \
   For example: dokon.bito.uz \
   (Each organization has a separate address.)
2. Enter your phone number
3. Enter your password
4. If the "Remember me" button is checked, you will be logged in automatically for 30 days. After 30 days, you will need to enter your login and password again.
5. Click the "Login" button

If the information is correct, you will successfully log into the system.

<figure><img src="../../.gitbook/assets/image (162).png" alt=""><figcaption></figcaption></figure>





### Password recovery

If you have forgotten your password, you can reset it independently.

1. Open the system login page (for example: dokon.bito.uz).
2. Click the **"Forgot your password?"** button.
3. Enter your phone number.
4. A verification code will be sent to your number via SMS.
5. Enter the SMS code.
6. Set a new password.
7. Log back into the system with the new password.

### Login via one-time password (SMS)

In the BITO system, users can log in using a **one-time SMS code** instead of entering a password.

* Go to the "**One-time password**" window
* Enter your **phone number** registered in the system.
* Click the **SMS icon** next to the phone number field.
* A one-time verification code will be sent via SMS to the entered number.
* Enter the received SMS code in the designated field.
* Click the **"Login"** button.
* If the code is correct, you will successfully log into the system.
