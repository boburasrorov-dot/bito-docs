# Profile Settings

<details>

<summary>Editing Personal Information</summary>

To access the profile window, click on the username located in the upper right corner of the system. From the opened menu, select the **Profile** section.

In the Profile section, you can:

* Upload your photo
* Edit your first and last name
* Change your phone number
* Date of birth
* INN
* Telegram
* Email

Enter and save the information

<summary>Changing Password (while logged in)</summary>

After logging into the system, users can update their password.

**Password change procedure:**

1. In the profile window, select the **Change Password** section.
2. Enter the current password.
3. Enter the new password.
4. Confirm the new password.
5. Click the **Save** button.

For security purposes, it is recommended to use a strong password.

<summary>Changing PIN Code (for mobile application)</summary>

A PIN code is used for quick access through the mobile application.

#### PIN code change procedure:

1. In the profile window, select the **PIN Code** section.
2. Enter a new 6-digit PIN code.
3. Click the "Save" button.

Note: The PIN code is only used for accessing the mobile application.

<details>

<summary>Lock</summary>

When the **Lock** button in the profile menu is pressed, the system is temporarily blocked.

* The screen is locked.
* You are not logged out of the system.
* Password entry is required to access again.

This function ensures security when the user temporarily leaves the computer.

<details>

<summary>Logout</summary>

When the **Logout** button is pressed, the user completely exits the system.

* The session is terminated.
* Login and password are required to access again.
