# Write-off Reasons



### Write-off Reasons

The Write-off Reasons section maintains a list of reasons used when writing off products from the warehouse. These reasons are subsequently selected during the write-off process and clearly indicate why a product was removed from the warehouse.

<figure><img src="../../.gitbook/assets/image (367).png" alt=""><figcaption></figcaption></figure>

The user navigates to **Warehouse → Write-off Reasons** section and clicks the **"Create"** button to add a new reason.

In this section, standard reasons for writing off products are created and managed.

For example:

* Defective product
* Loss
* Expiration date reached
* Damage

When creating a new reason, the following information is entered:

Name — write-off reason name\
Code — internal identification code (optional)\
Order number — determines the display order in the list\
Description — additional information (optional)\
Active status — whether the reason is active or inactive
