<summary>Creating a Discount</summary>

When accessing the **Discount** page in the Marketing section, a list of all discounts created in the system is displayed. Through this section, you can view existing discounts, edit them, and create new discounts.

By creating a discount, you can apply discounts automatically or manually during the sales process. For example, if a 10% discount is created and automatic assignment is enabled, this discount will be automatically applied to every sale. This eliminates the need to manually enter the discount each time and speeds up the sales process.

To create a new discount, click the Create button and enter the following information:

* **Name** — enter the discount name, used to distinguish and identify it in the system.
* **Type** — select the discount type (percentage or amount), which determines how the discount is calculated.
* **Amount** — enter the discount value (e.g., 10% or a specific amount), which shows how much discount is applied to the price.
* **Currency** — indicates in which currency the discount is calculated.
* **Duration** — select the validity period of the discount (permanent or temporary).
* **Start Date** — if temporary is selected, indicates when the discount begins to take effect.
* **End Date** — specifies when the discount expires.
* **Assignment Type** — determines whether the discount applies to all products or only selected products.
* **Products** — if specific is selected, choose the products to which the discount applies.
* **By Category** — allows selecting products by category.
* **Auto Assign** — when enabled, the discount is automatically applied during sales.
* **Manual Assignment** — allows the user to manually apply the discount.
* **Upload File** — allows attaching additional documents related to the discount.

After entering all the information, the discount is saved and ready to be used in the sales process.
