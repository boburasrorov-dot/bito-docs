# Cash Register Session



### Cash Register Session

The cash register session section manages and monitors the cash register's daily or specific time period operations. This section enables opening and closing the cash register, as well as analyzing all financial transactions conducted during the session.

To view cash register sessions, users access the **Finance → Cash Register Session** section from the left menu.

The cash register session section serves to monitor cashier activities, analyze cash flows, and identify discrepancies at the end of sessions.

#### General Indicators

This section displays the overall status of sessions for the selected period:

* **Total** — total number of sessions
* **Opened** — number of currently active (open) sessions
* **Closed** — number of completed sessions
* **Flagged** — sessions requiring attention or review
* **Total Difference** — sum of discrepancies identified across all sessions
* **Total Revenue** — total income generated during sessions

<figure><img src="../../.gitbook/assets/image (449).png" alt=""><figcaption></figcaption></figure>

#### Sessions List

This section displays each session in detail in a separate row:

* **Session ID** — unique session number
* **Opened Time** — session start time
* **Closed Time** — session end time
* **Cashier** — employee who managed the session
* **Cash Register** — which cash register it belongs to
* **Opening Balance** — cash amount at session start
* **Closing Amount** — final amount when session closed
* **Total Difference** — discrepancy between opening and closing amounts
* **Status** — session status (open, closed, etc.)

<figure><img src="../../.gitbook/assets/image (451).png" alt=""><figcaption></figcaption></figure>

#### Example:

If a cashier opens the register at the beginning of the day with 1,000,000 UZS and sales are conducted throughout the day, the system calculates the final amount when the session closes. If there is a difference between the actual cash and the system amount, it is shown as a discrepancy and allows for review.
