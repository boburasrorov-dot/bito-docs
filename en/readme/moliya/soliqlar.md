# Taxes

<details open>

<summary>Taxes</summary>

The Taxes section is used to create and manage tax rates applied to products and services in the system. This section serves to automate pricing, ensure accurate payment calculations, and maintain precision in financial reporting.

To add a tax, the user clicks the **"Create"** button and enters the required information.

<figure><img src="../../.gitbook/assets/image (421).png" alt=""><figcaption></figcaption></figure>

<figure><img src="../../.gitbook/assets/image (460).png" alt=""><figcaption></figcaption></figure>

#### Basic Information

At this stage, the main parameters of the tax are defined:

* **Name** — tax name (e.g., VAT, Excise, and others)
* **Rate (%)** — tax percentage rate, this value determines what percentage is added to the product price
* **Type** — tax type (e.g., included or other type), this defines how the tax is applied

<figure><img src="../../.gitbook/assets/image (461).png" alt=""><figcaption></figcaption></figure>

#### Application Settings

Here is where you define how the tax operates:

* **Manual assignment** — if enabled, the tax is manually selected in each transaction and is not applied automatically
* **Included in price** — determines whether the tax is automatically added to the product price

<figure><img src="../../.gitbook/assets/image (462).png" alt=""><figcaption></figcaption></figure>

#### Scope of Application

Select which products the tax applies to:

* **All products** — the tax applies to all products
* **Some products** — applies only to selected products (selected through the list below)

#### Linking with Products

If the tax applies only to certain products, the following options are available:

* search and add products
* select by category
* filter by supplier
