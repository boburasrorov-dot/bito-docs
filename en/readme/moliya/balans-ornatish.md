# Balance Setup

<details>

<summary>Customer Balance Setup</summary>

This section allows manual entry and comparison of balances for customers. This process is used to adjust accounts and align them with the actual state.

During the balance setup process, the user first selects the required customer and the system automatically displays their existing (plan) balance. Then the user enters the "actual" value according to the real state. The system compares this entered value with the existing balance and automatically calculates the difference between them. If a difference exists, this variance indicates the need to adjust the balance. After all data is entered, the balance is updated through saving and the accounts in the system are aligned with the actual state.

<figure><img src="../../.gitbook/assets/image (414).png" alt=""><figcaption></figcaption></figure>

<figure><img src="../../.gitbook/assets/image (34).png" alt=""><figcaption></figcaption></figure>

To set up a balance, the **Create** button is clicked and a list of customers appears in a special window.

The following columns are available for each customer:

* **Name** – customer name
* **Plan** – system (calculated) balance
* **Actual** – real (manually entered) balance
* **Difference** – variance between plan and actual

Balances are entered in the following currencies:

* **Soum**
* **USD (and other currencies may be available)**

#### Data Entry

* The actual balance is entered in the **Actual** column
* The system automatically calculates the **Difference**
* At the bottom, total **Summary** indicators are displayed

#### Saving

* **Save** – saves the entered balances
* **Cancel** – cancels the operation
