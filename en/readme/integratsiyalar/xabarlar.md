# Messages

<details open>

<summary>Creating a message template</summary>

The Messages section allows you to create and manage templates for messages sent to customers and users. Through this section, pre-prepared texts are stored and used automatically or manually in necessary processes.\
This section is used to speed up the messaging process, maintain consistent formatting, and standardize communication.

<figure><img src="../../.gitbook/assets/image (1).png" alt=""><figcaption></figcaption></figure>

Navigate to Integrations → Messages section and create a new message template through the **"Create"** button. Message text is entered in each template and saved for later use in various processes.

This section consists of two main parts:

* **Message template** – list of created messages
* **Message history** – history of sent messages

Message templates provide the following capabilities:

* Use ready-made texts for quick messaging
* Automate sending the same messages without rewriting them repeatedly
* Create separate templates for different situations (debt notifications, reminders, and others)

Through this section, you can optimize the customer communication process and systematically manage message sending.

</details>
