# Integrations

<details open>

<summary>Integrations</summary>

The Integrations section allows you to connect the system with external services and platforms. Through this section, integrations with payment systems, banks, telephony services, and other services are established, and all processes are managed in a unified system.\
This section is used to automate business processes, accelerate data exchange, and connect additional services to the system.

<figure><img src="../../.gitbook/assets/image.png" alt=""><figcaption></figcaption></figure>

Access Integrations → Integrations section, select the required service, and activate the integration through the **"Install"** button. For some integrations, API keys or additional settings may be required.

In this section, all integrations are displayed in card view. Through each card, you can view and manage the integration status.

Available integrations are categorized as follows:

* Payment systems (Click, Payme, BITO Pay, and others)
* Banking and terminal services (Uzum Bank, Marta Terminal)
* Telephony (Online PBX)
* Other services (CRM, HRM, OFD, etc.)

The following features are available for each integration:

* Install or disable the integration
* View status (installed / not installed)
* Manage settings
* Real-time data exchange with the system

Through this section, you can automatically accept payments, synchronize with external systems, and accelerate operations.
